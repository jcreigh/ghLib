ghLib is a C++ helper library created by The Gearheads (Team 1189) for use with FRC.

Installation

  1. Download the newest release from Releases
  2. Simply extract to the wpilib location (e.g. C:\Users\Gearhead\wpilib)
  3. The new (2017) WPILib Eclipse plugins will automatically locate and use the library and headers
